# 🪨📄✂️ Rock-Paper-Scissors Game

A simple and interactive **Rock-Paper-Scissors** game built using **HTML, CSS, and JavaScript**.  
The user selects Rock, Paper, or Scissors, and the computer randomly picks its move.  
Scores update automatically, and a reset button lets you restart the game anytime.

---

## 🚀 Features

- 🎮 Play Rock-Paper-Scissors against the computer  
- 🤖 Computer generates a random choice each round  
- 🟩 Win, 🟥 lose, or 🟧 tie messages with color feedback  
- 🔢 Live score updates  
- 🔄 Reset button to restart the game  
- ⚡ Fast, responsive, and beginner-friendly code  

---

## 🧠 How It Works

- The user clicks one of the three choices  
- The computer randomly selects its move
- JavaScript compares the user's choice with the computer's choice
- The winner is determined using standard RPS rules  
- Scores update dynamically  
- The reset button clears everything back to zero
